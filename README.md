#SmartZoom

**Sketch**
-----------------

由两个Activity组成的头像缩放Demo，并且加入了背景Blur效果。不具备构成类库的条件，只是众多实现形式之一，没有投入复杂的代码与逻辑，因此它看起来，并不是那么漂亮。
